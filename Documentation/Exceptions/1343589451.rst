==========================
TYPO3 Exception 1343589451
==========================

.. include:: /If-you-encounter-this-exception.rst.txt

Could not fetch page data for uid "...".
=========================================

Error in frontend when using for example :html:`<f:link.page>` ViewHelper because
the pageUid was a deleted page. Make sure the target of the link is accessible.
